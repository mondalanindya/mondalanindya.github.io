<h1 id="projects">Projects <a href="https://sites.google.com/view/anindyamondal/research-projects"></a></h1>
<ul>
<li><p><strong>Source-free Domain Adaptation</strong></p>
<p><em>Working with Prof. Soma Biswas</em></p>
<p>Currently using source-free domain adaptation to perform pixel-level tasks in computer vision.</p></li>
<li><p><strong>Reconstructing Time-varying Graph Signals using Graph Neural Networks</strong></p>
<p><em>Working with Jhony H. Giraldo and Prof. Thierry BOUWMANS</em></p>
<p>Currently working on designing a Graph Convolutional Network (GCN) framework to reconstruct time-varying graph signals from their samples.</p></li>
<li><p><strong>Recovery of Missing Data in IoT Sensor Networks using Graph Signal Processing</strong></p>
<p><em>Advised by Prof. P. Venkateswaran (Bachelor Project - Jadavpur University)</em></p>
<p>Applied a Sobolev norm minimization method for reconstructing time-varying graph signals to recover missing data in sensor networks. Our approach improved upon the previous works by as much as 54%. Paper accepted at <strong>EUSIPCO 2022</strong>.</p></li>
<li><p><strong>Graph-based Approaches for Moving Object Detection and Semantic Segmentation for Event-based Vision</strong></p>
<p><em>Worked with Prof. Ananda S. Chowdhury, Thierry BOUWMANS, and Jhony H. Giraldo</em></p>
<p>Worked on developing a new algorithm based on Graph Spectral Clustering to detect moving objects in a scene captured by an event-based camera and presented the results at the <strong>ICCV Workshops 2021</strong></p>
<p>Worked on developing a graph-based semi-supervised learning algorithm for semantic segmentation in event-based data.</p></li>
</ul>
